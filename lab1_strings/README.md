# Лабораторная работа №1

## Тема
Работа с Git и GitHub. Работа со строками на C++.

## Индивидуальное задание
Приведение всех букв строки к верхнему и нижнему регистру.

## Описание программы
Программа принимает строку с клавиатуры и по выбору пользователя:
- переводит строку в верхний регистр;
- переводит строку в нижний регистр.

## Технологии
- C++
- VS Code
- Git
- GitHub
